BROWSE FASTER

ZunoZap is a lite & fast web browser made for you. (Credits)

Unlike other browsers ZunoZap uses a average of 400MB of RAM

(Chrome 1GB, Edge 2GB, Firefox 700MB).


ZunoZap includes:


- Addon API (Plugins & themes)

- Settings.

- Auto-updating

- Offline web site viewer (beta)

- Web engine switcher. (Chromium, WebKit)

- Lite as a feather.

ZunoZap was created on Jul 6, 2016.

ZunoZap v0.1.0, the first beta release, was published on Mar 19, 2017.
