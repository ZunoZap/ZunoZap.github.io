ZunoZap

A Simple Java web browser!

ZunoZap (zoo-u-no-z-ah-p) is A simple web browser coded in Java

ZunoZap is in HIGHY DEVELMENT! Do not use as your main browser yet!

ZunoZap can do lots of things that most web browsers can't! like:
ZunoZap lets you edit any website with out knowing HTML
ZunoZap lets you download a website's source
ZunoZap lets you view any website offline (Allmost Done)
